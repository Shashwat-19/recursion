C++ Implementation

int fib(int n) {
    // Base Case
    if (n <= 1) 
        return n;
    
    // Recursive Case
    return fib(n-1) + fib(n-2);
}
Key Concept:

Every time a function calls itself, the computer pushes a Stack Frame onto the memory stack to save local variables and the return address.

1. What is Recursion?

Recursion is a technique where a function calls itself to solve smaller instances of the same problem.

  • Base Case: The condition to stop recursion (e.g., if n <= 1).
  • Recursive Step: The logic where the function calls itself.

2. Advantages & Disadvantages

Advantages:
  • Code is cleaner and more concise.
  • Excellent for tree/graph traversals (DFS).
  • Naturally solves problems like Tower of Hanoi.
Disadvantages:
  • Stack Overflow: If depth is too high, memory runs out.
  • Overhead: Function calls are slower than loops (iteration).

3. Applications

  • Data Structures: Trees, Graphs (DFS).
  • Algorithms: QuickSort, MergeSort, Binary Search.
  • Parsing: Compilers use recursion to parse syntax.
  • Mathematics: Factorials, Fibonacci Series.
